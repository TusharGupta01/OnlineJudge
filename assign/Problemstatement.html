We are making a platform basically an online judge. This is a platform where user can submit his/her code and we will be providing them detailed report of their code. We are providing feature, they are as follows:
1. Whether code compiled successfully and if not report the compile-time error.
2. Whether the program generated any run time error.
3. Whether each test case was passed successfully.
4. Time taken to pass each test case.
5. Total time and space taken by the program.
6. Plagiarism Checker
